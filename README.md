# A brief description about this project
This school project is a quiz game that i've work tohether with a group of five people. The quiz is a about flags and what country each flag represent. It's ten questions and you have three anwers to choose from, wich one you think it's right. You can follow how many questions you have answerd and how far you have come in the process with the car at the bottom of the page that moves forward for each answerd question. When the quiz is finished the result will show have many qustions you answerd correct and time, also an animation pops up with confetti! ðŸŽ‰
